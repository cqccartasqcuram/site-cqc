Nossa Missão

Projeto Cartas q Curam é um projeto onde enviamos cartas de motivação e de gratidão, para as pessoas que estão com transtornos psíquicos (depressão, síndrome do pânico, ansiedade, etc.) como também, encarcerados, alcoólatras, moradores de rua, enlutados, idosos, etc.
As cartas são de próprio punho, enviadas via Correios para pessoas nunca dantes vistas. Esta é uma das formas encontradas para dizer ao mundo que as palavras curam.
